Float block elements left and right

Sometimes you want to float (also called pull) an element either all the way to the left or all the way to the right of its container element.

Once you do that, the following elements will be crammed in between the floated ones. You can use clearfix on the parent of the floated elements to make sure the following elements, like this text, appear below them.
